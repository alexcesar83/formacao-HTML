Ferramentas Utilizadas

(Voltar)

Durante o curso utilizaremos a ferramenta VS Code.

O Visual Studio Code (VS Code) é um editor de código-fonte desenvolvido pela Microsoft. Ele é gratuito, de código aberto e altamente personalizável, sendo amplamente utilizado por desenvolvedores em diversas plataformas.

    Algumas de suas principais características incluem:
  1. Multiplataforma: Disponível para Windows, macOS e Linux
  2. Extensibilidade: Suporta uma ampla variedade de extensões para adicionar funcionalidades adicionais, como suporte a diferentes linguagens de programação, integração com ferramentas de desenvolvimento, temas e muito mais.
  3. Integração com Git: Oferece integração integrada com o Git, permitindo controle de versionamento e gerenciamento de repositórios diretamente no editor.
  4. Depuração Integrada: Possui recursos de depuração integrados para várias linguagens de programação, permitindo aos desenvolvedores depurar seus aplicativos diretamente no editor.
  5. Terminal Integrado: Inclui um terminal integrado que permite executar comandos diretamente no editor, facilitando o fluxo de trabalho do desenvolvedor.
  6. Autocompletar Inteligente: Oferece recursos de autocompletar inteligentes que ajudam os desenvolvedores a escrever código mais rapidamente e com menos erros.
  7. Customização: Permite personalizar praticamente todos os aspectos do editor, desde temas e atalhos de teclado até configurações de formatação de código.

Inspetor de Elementos

(Voltar)

O inspetor de elementos é uma ferramenta encontrada em navegadores da web que permite aos desenvolvedores visualizar e editar o código HTML, CSS e JavaScript de uma página da web em tempo real. É amplamente utilizado para depurar e desenvolver websites, oferecendo funcionalidades como edição de HTML e CSS, depuração de JavaScript e análise de desempenho da página.

Para acessar o inspetor de elementos em um navegador da web, clique com o botão direito do mouse em qualquer parte da página e selecione "Inspecionar" ou "Inspecionar elemento" no menu de contexto. Você também pode pressionar a tecla F12 (ou Ctrl+Shift+I) para abrir o inspetor de elementos.


Estrutua Básica do HTML

(Voltar)

A estrutura básica do HTML consiste em um documento que começa com a declaração "< !DOCTYPE html >", seguida por uma tag "< html >" que envolve todo o conteúdo da página. O "html" contém duas seções principais: "< head >", que inclui metadados como título, codificação de caracteres e configurações de visualização, e "< body >", onde está o conteúdo visível da página, como texto, imagens e formulários.


Falando sobre Tags

(Voltar)

Em HTML, uma tag é um elemento de marcação usado para definir diferentes partes de um documento, como texto, imagens, links, entre outros. As tags HTML são cercadas por colchetes angulares < > e geralmente aparecem em pares, com uma tag de abertura e uma tag de fechamento. A tag de abertura define o início de um elemento, enquanto a tag de fechamento marca o final desse elemento. Por exemplo, < p > é a tag de abertura para um parágrafo, enquanto < /p > é a tag de fechamento. O conteúdo que está entre essas tags é o que será exibido na página da web. Algumas tags, como para imagens ou para links, não requerem uma tag de fechamento e podem ser usadas sozinhas. As tags HTML são a base para estruturar e organizar o conteúdo de uma página da web.


Atributos das Tags

(Voltar)

Atributos em HTML são informações adicionais dentro das tags que modificam ou definem o comportamento dos elementos. Eles consistem em um nome e um valor, separados por um sinal de igual, e são usados para especificar detalhes como o destino de um link, o caminho de uma imagem ou classes CSS para aplicação de estilos. Os atributos são cruciais para personalizar e controlar o comportamento dos elementos em uma página da web.


Textos

(Voltar)

Em HTML os textos são os conteúdos visíveis que são exibidos na página da web. Eles podem ser simplesmente texto puro ou podem estar incluídos dentro de elementos HTML, como parágrafos, títulos, listas, entre outros. Os textos podem incluir caracteres alfanuméricos, símbolos e espaços em branco e são essenciais para comunicar informações aos usuários em uma página da web.


Listas Ordenadas e Não Ordenadas

(Voltar)

Listas ordenadas em HTML são numeradas sequencialmente, enquanto listas não ordenadas são marcadas com símbolos como pontos ou círculos. As listas ordenadas são úteis para sequenciar passos ou elementos importantes, enquanto as listas não ordenadas são adequadas para itens sem uma ordem específica, como recursos ou pontos de destaque. Esses elementos são fundamentais para organizar e apresentar informações de forma estruturada em páginas da web.


Links

(Voltar)

Links em HTML são criados com a tag < a >. Eles permitem a navegação entre diferentes páginas da web ou dentro da mesma página. Os links podem ter um endereço URL como destino ou um identificador de âncora para navegação interna. São fundamentais para conectar diferentes recursos online e facilitar a navegação e acessibilidade na web.
